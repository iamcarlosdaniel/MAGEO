SIGEMUV (Sistema de Gesti√≥n Integral de Eventos y Marketing de la Universidad del Valle) is a platform developed to comprehensively manage upcoming events, as well as coordinate with the Marketing and Maintenance departments to efficiently and effectively carry out the necessary tasks for these events.

### Quick start

1. Clone the repository to your local machine:

   ```sh
   git clone https://github.com/iamcarlosdaniel/SIGEMUV
   ```

2. Navigate to the project directory:

   ```sh
   cd SIGEMUV
   ```

3. Install the necessary dependencies:

   ```sh
   npm install
   ```

4. Start the development server:

   ```sh
   npm run dev
   ```
